# demos

命名规范:

全小写，中划线

关于文件夹的类型以后通过前标进行标注:

+ css-
+ js-


CDN引用说明，大部分引用http://www.bootcdn.cn/，更新的比较快，比较稳定

+ zepto.js ://cdn.bootcss.com/zepto/1.2.0/zepto.min.js
+ jquery ://cdn.bootcss.com/jquery/3.0.0/jquery.js
+ css://cdn.bootcss.com/bootstrap/3.3.7/css/bootstrap.min.css
